    Installation and setup

    1. sudo apt-get install postgresql
    2. sudo su postgres
    3. sudo createuser -s vagrant
    4. exit
    5. createdb vagrant
    6. wget http://www.sportsdb.org/modules/sd/assets/downloads/sportsdb_sample_postgresql.zip
    7. unzip sportsdb_sample_postgresql.zip
    8. psql < sportsdb_sample_postgresql_20080304.sql

    We are now ready! We now have a database that we can use and play around with! 0. psql 1. Get a list of the tables using \dt. There are a lot of tables that came from sportsdb_sample_postgresql_20080304.sql! 2. What is the postgres command to get the information/schema about the following tables? Run that command on the table american_football_action_participants to the column names and the data types. 3. Write a select statement that returns the yardage of all the players in the table american_football_action_participants. 4. Write a select statement that returns the birth_date of all the persons in the tables persons. 5. Now write a select statement with a join that returns the birth_date and yardage using both the persons and american_football_action_participants tables. In other words, join on the person_id.
